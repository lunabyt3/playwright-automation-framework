# Playwright Automation Framework

UI Automation Testing Project using Playwright and JavaScript.

## Project Objective

Automate login validation for web applications.

## Test Scenario

- Open Login Page
- Enter Username
- Enter Password
- Click Login
- Verify Successful Login

## Tools Used

- Playwright
- JavaScript
- Node.js
- Git

## Execute Test

```bash
npx playwright test
```

## Result

Test execution completed successfully.
